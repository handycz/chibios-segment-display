uint8_t segdisp::flags

Configuration flags. For common anode and cathode configuration use SEGDISP_COMMON_ANODE and SEGDISP_COMMON_CATHODE respectively. To configure NPN and PNP driver, use SEGDISP_NPN_DRIVER and SEGDISP_PNP_DRIVER constants. For 7 segment display use SEGDISP_SEGMENTS_SEVEN, for 16 segment display use SEGDISP_SEGMENTS_SIXTEEN
